## 📝 Overview

**Wordly** is a modern, responsive **Single Page Application (SPA)** that serves as an interactive dictionary tool. It allows users to search for words, view detailed definitions, listen to pronunciations, explore synonyms, and save favorite words for future reference — all without ever refreshing the page.

Built with vanilla JavaScript, CSS3, and HTML5, Wordly leverages the **Free Dictionary API** to provide accurate and comprehensive word data in real-time. The application demonstrates proficiency in asynchronous JavaScript, DOM manipulation, localStorage persistence, and SPA architecture.

---

## ✨ Features

### Core Functionality
- 🔍 **Word Search** - Instant word lookup with real-time results
- 📖 **Definitions** - Multiple meanings with parts of speech
- 🔊 **Audio Pronunciation** - Listen to word pronunciations (when available)
- 📝 **Example Sentences** - Contextual usage examples
- 🔗 **Synonyms** - Related words for expanded vocabulary
- ⭐ **Favorites** - Save and manage favorite words using localStorage
- 🎯 **Click-to-Search** - Click any favorite word to search it again

### User Experience
- ⚡ **SPA Architecture** - No page reloads, seamless navigation
- 🌙 **Dark Theme** - Modern dark interface with purple accents
- 📱 **Responsive Design** - Optimized for desktop, tablet, and mobile
- ♿ **Accessibility** - Keyboard navigation and screen reader support
- ⏱️ **Loading States** - Visual feedback during API requests
- ❌ **Error Handling** - Clear, user-friendly error messages
